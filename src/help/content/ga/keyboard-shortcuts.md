# Aicearraí méarchláir

Coimeád do lámha ar an méarchlár. Ar macOS úsáid **⌘**; ar Windows agus Linux úsáid **Ctrl**.

## Ag breathnú go gairid

- **⌘K** — oscail Cabhair Tapaidh ó áit ar bith.
- **⌘\** — scoránaigh mód fócais.
- **Esc** — dún forlaigh nó cealaigh eagarthóireacht.
- **Ctrl/⌘ + Enter** — sábháil toghchán atá á chur in eagar agat.

## Uilíoch

- **⌘K** — oscail an forlaigh **Cabhair Tapaidh** ó áit ar bith. Cuardaigh an Help Center
  nó léim go dtí aicearra, ansin brúigh **Esc** chun dúnadh.
- **⌘\** — scoránaigh [mód fócais](views-and-modes#focus) ar siúl agus as.

## Markdown eagarthóra

Clóscríofa ag tús líne, claochlaíonn iad seo agus tú ag scríobh:

- `# ` / `## ` / `### ` — ceanntásca
- `- ` / `* ` — liosta bullae
- `1. ` — liosta uimhrithe
- `> ` — bloc-luachan

Cuimsiú focal le `**` le haghaidh trom nó `*` le haghaidh iodálach.

## Phána toghcháin

- **Esc** — dún an phána (nó cealaigh eagarthóireacht atá ar siúl)
- **Ctrl/⌘ + Enter** — sábháil an iontrál atá á chur in eagar agat

## Nóta faoi aicearraí brabhsálaí

Tá roinnt teaglamaí eochracha in áirithe ag do bhrabhsálaí nó córas oibriúcháin. Nuair a
shainmhíníonn an aip aicearra, glacann sé tosaíocht ar a scáileáin féin; in áit eile
bíonn réamhshocrú an chórais i réim.
