# Formáidiú & markdown

Formáidigh téacs ar thrí bhealach: an barra uirlisí snámhach, aicearraí méarchláir, nó
giorrúcháin markdown clóscríofa agus tú ag scríobh.

## Ag breathnú go gairid

- Clóscríobh `## ` le haghaidh ceanntásc, `- ` le haghaidh bullae, `1. ` le haghaidh liosta uimhrithe.
- Cuimsiú focal le `**trom**` nó `*iodálach*`.
- Roghnaigh téacs chun an barra uirlisí snámhach a úsáid ina ionad.

## Aicearraí markdown

Clóscríobh iad seo ag tús líne (nó timpeall focal) agus claochlaíonn siad go huathoibríoch:

- `# ` → ceanntásc 1, `## ` → ceanntásc 2, `### ` → ceanntásc 3
- `- ` nó `* ` → liosta bullae
- `1. ` → liosta uimhrithe
- `> ` → bloc-luachan
- `**trom**` → **trom**, `*iodálach*` → *iodálach*

## Formáidiú inlíne

Roghnaigh téacs agus úsáid an barra uirlisí snámhach, nó úsáid an méarchlár:

- **Trom** — an t-aicearra trom
- *Iodálach* — an t-aicearra iodálach

## Ceanntásca agus struchtúr

Sainmhíníonn ceanntásca an **imlíne** cáipéise a thaispeántar sa chigire. Úsáid iad chun
píosaí fada a choinneáil inléimthe.

## Liostaí agus luachanna

Coinníonn liostaí agus bloc-luachanna a bhformáidiú nuair a bhrúnn tú Enter, mar sin is féidir
leat leanúint ag clóscríobh gan an luch a aimsiú. Brúigh Enter ar mhír liosta folamh chun an
liosta a fhágáil.
