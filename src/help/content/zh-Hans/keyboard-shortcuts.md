# 键盘快捷键

保持双手在键盘上。macOS 使用 **⌘**；Windows 和 Linux 使用 **Ctrl**。

## 概览

- **⌘K** — 从任意位置打开快速帮助。
- **⌘\** — 切换专注模式。
- **Esc** — 关闭浮层或取消编辑。
- **Ctrl/⌘ + Enter** — 保存正在编辑的引用条目。

## 全局快捷键

- **⌘K** — 从任意位置打开**快速帮助**浮层。搜索 Help Center 或跳转到快捷键，然后按 **Esc** 关闭。
- **⌘\** — 切换[专注模式](views-and-modes#focus)的开启与关闭。

## 编辑器 Markdown

在行首输入以下内容，输入时自动转换：

- `# ` / `## ` / `### ` — 标题
- `- ` / `* ` — 项目符号列表
- `1. ` — 编号列表
- `> ` — 块引用

用 `**` 包裹单词表示粗体，用 `*` 表示斜体。

## 引用面板

- **Esc** — 关闭面板（或取消正在进行的编辑）
- **Ctrl/⌘ + Enter** — 保存正在编辑的条目

## 关于浏览器快捷键

部分按键组合被浏览器或操作系统保留。当应用定义了快捷键时，它在自己的页面中优先生效；其他位置则遵从系统默认。
