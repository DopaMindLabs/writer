# 格式与 Markdown

格式化文本有三种方式：浮动工具栏、键盘快捷键，或边写边使用的 Markdown 快捷输入。

## 概览

- 输入 `## ` 创建标题，`- ` 创建项目符号，`1. ` 创建编号列表。
- 用 `**粗体**` 或 `*斜体*` 包裹单词。
- 选中文本后使用浮动工具栏同样可以格式化。

## Markdown 快捷键

在行首（或围绕单词）输入以下内容，会自动转换：

- `# ` → 一级标题，`## ` → 二级标题，`### ` → 三级标题
- `- ` 或 `* ` → 项目符号列表
- `1. ` → 编号列表
- `> ` → 块引用
- `**粗体**` → **粗体**，`*斜体*` → *斜体*

## 内联格式

选中文本后使用浮动工具栏，或使用键盘：

- **粗体** — 粗体快捷键
- *斜体* — 斜体快捷键

## 标题与结构

标题定义检查器中显示的文档**大纲**。利用它来让长篇文章保持可导航性。

## 列表与引用

列表和块引用在按下 Enter 键时保持格式，因此无需使用鼠标即可继续输入。在空列表项上按 Enter 可退出列表。
