# Tastenkürzel

Halte deine Hände auf der Tastatur. Auf macOS verwende **⌘**; auf
Windows und Linux verwende **Strg**.

## Auf einen Blick

- **⌘K** — öffne Schnellhilfe von überall.
- **⌘\\** — Fokusmodus umschalten.
- **Esc** — eine Überlagerung schließen oder eine Bearbeitung abbrechen.
- **Strg/⌘ + Enter** — speichere ein Zitat, das du bearbeitest.

## Global

- **⌘K** — öffne die **Schnellhilfe**-Überlagerung von überall. Suche im Hilfe-Center oder springe zu einem Kürzel, dann drücke **Esc** zum Schließen.
- **⌘\\** — schaltet den [Fokusmodus](views-and-modes) um.

## Editor-Markdown

Am Zeilenanfang getippt, transformieren sich diese, während du schreibst:

- `# ` / `## ` / `### ` — Überschriften
- `- ` / `* ` — Aufzählungsliste
- `1. ` — nummerierte Liste
- `> ` — Blockzitat

Umschließe ein Wort mit `**` für fett oder `*` für kursiv.

## Zitate-Panel

- **Esc** — schließe das Panel (oder breche eine laufende Bearbeitung ab)
- **Strg/⌘ + Enter** — speichere den Eintrag, den du bearbeitest

## Hinweis zu Browser-Kürzeln

Einige Tastenkombinationen sind von deinem Browser oder
Betriebssystem reserviert. Wo die App ein Kürzel definiert, hat es
auf ihren eigenen Bildschirmen Vorrang; an anderer Stelle gewinnt der
Systemstandard.
