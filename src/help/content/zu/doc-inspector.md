# Umhlolelo wedokhumenti

Umhlolelo wedokhumenti iphaneliyu eceleni komhleli. Ithebhu yayo ye-**Uhlaka** ibeka uhlelo lwedokhumenti, nethebhu yayo ye-**Ulwazi** ibonisa imininingwane ephilayo ngeidokhumenti obhala kuyo futhi ikuvumele usethe izivimbelo nezimo ezingeziwe. Wena ukhetha ukuthi ziphi zalezi zinkambu ezivela — ngokubanzi nangenaneli ngalinye — ngakho-ke umhlolelo uveza kuphela lokho okukuqukethwe wena.

## Uhlaka

Ithebhu ye-**Uhlaka** ihlela izihloko zedokhumenti ngokulandelana, ileliwe ngezinga, ukuze ukwazi ukubona isimo sendaba ende ngokuhlola. Ivuselelwa ngokufushane ngemuva kokuphumula uthayiping.

Uhlaka lwakhiwa ngezihloko zakho — thayipha `# `, `## ` noma `### ` ekuqaleni komugqa (noma usebenzise igude eliwayo) ukuengeza elinye. Idokhumenti engenazisihloko ibonisa uhlaka olude; bheka ku-[Ukuhlobisa ne-markdown](formatting-and-markdown#headings-and-structure) ngendlela izihloko ezisebenza ngayo.

Kufonini, vula umhlolelo kusuka ku-ithebhu ye-**okunye**: khetha **Umhlolelo wedokhumenti** kumenyu futhi uslayide uvela ngakwesokunene.

## Izibalo zamazwi nezinhlamvu

Ithebhu yeLwazi ihlala ibonisa izibalo ze-**mazwi** ne-**nhlamvu** eziphilayo zedokhumenti.

## Ukusetha umkhawulo wamazwi noma wezinhlamvu

Uma inkambu ye-**Umkhawulo wamazwi** noma ye-**Umkhawulo wezinhlamvu** ivezwa, thayipha inombolo ukusetha inhloso. Ukulanda kubala manje nge-`samanje / umkhawulo`, futhi uma weqa umkhawulo isibalo sishintsha sibe bomvu futhi umbhalo owedlula umkhawulo ukukhanyiswa kumhleli. Akukho okuyisuswa noma okugequlwa — ukukhanyisa kuphela uphawu lwamehlo, ngakho-ke ungaqhubeka ubhala bese ugunda kamuva.

Shiya umkhawulo ube yize (noma usethe ku-`0`) ukuze ungemkhawulo.

Uma ungathanda ukugcina umkhawulo kanye nokuphinda kodwa hhayi ukukhanyisa komhleli, vala **Khanyisa umbhalo owedlula umkhawulo** ezilungiselweni (bheka ngezansi).

## Isimo nokukhiya

Isikhombi **seSimo** sihambisa idokhumenti ezigabeni zaso: _Iseduli_, _Iyaqhubeka_, _Iphinde ihlolwa_, _Iqediwe_ ne-_Ishicilelwe_. Ukusetha isimo ku-**Iqediwe** noma **Ishicilelwe** kukhiya idokhumenti ukuze ingaguqulwe ngengozi — i-banner ivela ngaphezu komhleli inkinobho **Khulula ukuhlela**. Ukuze ukhulule, sebenzisa leyo nkinobho noma setha isimo sube yisigaba sangaphambilini. Ukukhiya kuvikela kuphela umzimba wedokhumenti; akukho okuyisuswa.

## Idethi yokufika

Uma inkambu ye-**Idethi yokufika** ivezwa, khetha idethi ukurekhoda umhlahlandlela. Idethi edlule ivezwa ngokubomvu.

## Ukukhetha ukuthi yiziphi izinkambu ezivela

Yonke inkambu yomhloleli iyinketho. Ukuze ukhethe ukuthi yiziphi ezivela:

- **Ngokubanzi:** vula **Izilungiselelo ezijwayelekile → Umhlolelo wedokhumenti** futhi vula noma uvale inkambu nganye. Ungakhetha nazo izikhombi zezilinganiso ezivela ku-isikhethi.
- **Ngokunaneli ngalinye:** vula **Izilungiselelo → Umhlolelo wedokhumenti** wenaneli. Inkambu nganye ingathatha izinto ezijwayelekile ezijwayelekile noma ivulwe noma ivalwe kulelo naneli kuphela.

Ukuvala inkambu kuyafihlela kumhloleli — kanye nomkhawulo wayo nokukhanyisa komhleli — ngisho uma ubekile inani ngaphambilini. Inani liyasuswa: buyisa inkambu futhi iphinde ivele.
