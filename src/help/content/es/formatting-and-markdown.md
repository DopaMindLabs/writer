# Formato y Markdown

Da formato al texto de tres maneras: la barra flotante, los atajos de teclado o
la sintaxis de Markdown que escribes en línea.

## De un vistazo

- Escribe `## ` para un encabezado, `- ` para una viñeta, `1. ` para una lista numerada.
- Envuelve una palabra en `**negrita**` o `*cursiva*`.
- Selecciona texto para usar la barra flotante en su lugar.

## Atajos de Markdown

Escribe estos al principio de una línea (o alrededor de una palabra) y se
transforman automáticamente:

- `# ` → encabezado 1, `## ` → encabezado 2, `### ` → encabezado 3
- `- ` o `* ` → lista con viñetas
- `1. ` → lista numerada
- `> ` → cita
- `**negrita**` → **negrita**, `*cursiva*` → *cursiva*

## Formato en línea

Selecciona texto y usa la barra flotante, o usa el teclado:

- **Negrita** — el atajo de negrita
- *Cursiva* — el atajo de cursiva

## Encabezados y estructura

Los encabezados definen el **esquema** del documento que se muestra en el
inspector. Úsalos para mantener navegables los textos largos.

## Listas y citas

Las listas y las citas en bloque conservan su formato al pulsar Enter, así
puedes seguir escribiendo sin coger el ratón. Pulsa Enter en un elemento vacío
para salir de la lista.
