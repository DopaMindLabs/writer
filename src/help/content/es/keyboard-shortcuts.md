# Atajos de teclado

Mantén las manos en el teclado. En macOS usa **⌘**; en Windows y Linux usa
**Ctrl**.

## De un vistazo

- **⌘K** — abre la Ayuda rápida desde cualquier sitio.
- **⌘\\** — alterna el modo enfoque.
- **Esc** — cierra una superposición o cancela una edición.
- **Ctrl/⌘ + Enter** — guarda una cita que estás editando.

## Globales

- **⌘K** — abre la superposición de **Ayuda rápida** desde cualquier sitio. Busca en el Centro de ayuda o salta a un atajo, y pulsa **Esc** para cerrar.
- **⌘\\** — alterna el [modo enfoque](views-and-modes).

## Markdown del editor

Escritos al principio de una línea, se transforman mientras escribes:

- `# ` / `## ` / `### ` — encabezados
- `- ` / `* ` — lista con viñetas
- `1. ` — lista numerada
- `> ` — cita

Envuelve una palabra con `**` para negrita o con `*` para cursiva.

## Panel de citas

- **Esc** — cierra el panel (o cancela una edición en curso)
- **Ctrl/⌘ + Enter** — guarda la entrada que estás editando

## Nota sobre los atajos del navegador

Algunas combinaciones de teclas están reservadas por tu navegador o sistema
operativo. Donde la app define un atajo, tiene prioridad en sus propias
pantallas; fuera de ellas, gana el atajo del sistema.
