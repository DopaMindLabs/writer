# Historial de versiones

Cada documento guarda su propio historial, así puedes mirar atrás cómo creció un
borrador y volver a una versión anterior cuando lo necesites. Como todo lo
demás, este historial se guarda localmente en tu navegador.

## Cómo se guardan las versiones

- **Versión inicial.** La primera vez que abres un documento, su estado actual queda registrado como punto de partida.
- **Automática.** Mientras escribes, el documento se guarda en segundo plano cada pocos minutos, pero solo cuando el texto ha cambiado.
- **Manual.** Guarda una versión tú mismo en cualquier momento desde la pestaña **Historial** o **Acciones** del inspector. Dale un nombre (por ejemplo «antes de la revisión») para que sea fácil reconocerla luego.

Las versiones automáticas se podan con el tiempo para no llenar el almacenamiento.
Las versiones que **fijas** y las que guardas **manualmente** nunca se eliminan
automáticamente.

## Ver y comparar versiones

Abre el inspector del documento y elige la pestaña **Historial** para ver la
lista, con la más reciente arriba. Selecciona **historial completo de versiones**
para abrir la vista de comparación, donde puedes elegir una versión anterior y
ver en qué se diferencia de tu texto actual. Alterna entre una vista de
diferencias **en línea** y una **lado a lado** para leer los cambios como
prefieras — el texto añadido y el eliminado se distinguen por color.

## Fijar una versión

Fija cualquier versión para protegerla de la limpieza automática — útil para
marcar un hito como un primer borrador terminado. Fíjala y desfíjala desde la
lista de versiones.

## Restaurar una versión anterior

Elige **Restaurar** en una versión para devolver el documento a ese estado.
Antes de sobrescribir el documento, tu texto actual se guarda como una nueva
versión automáticamente, así que una restauración siempre se puede deshacer.
