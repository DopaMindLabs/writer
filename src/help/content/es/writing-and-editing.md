# Escritura y edición

El editor está pensado para prosa larga.

## De un vistazo

- **Escribe** — todo se guarda al instante en tu navegador.
- **Selecciona texto** — aparece una barra flotante para formato rápido.
- **Renombra** — haz clic en el nombre del documento en la barra superior.
- **Enfoque** — pulsa **⌘\\** para ocultar todo menos tus palabras.

## Guardado automático

No hay botón de guardar ni riesgo de perder un borrador. Como el almacenamiento
es **local**, lo que escribes nunca sale de tu dispositivo a menos que lo
exportes — consulta [Tus datos](your-data).

## La barra flotante

Selecciona texto para revelar una pequeña barra flotante con el formato más
común. ¿Prefieres el teclado? Cada acción tiene también un atajo — consulta
[Formato y Markdown](formatting-and-markdown).

Puedes desactivar la barra flotante en
[Configuración rápida](customization-and-settings).

## Renombrar un documento

Haz clic en el nombre del documento en la barra superior para renombrarlo. Si
no lo nombras, se deduce un título a partir de tu primera línea para que la
barra lateral siga siendo legible.

## Mantener el foco

Pulsa **⌘\\** para cambiar al **modo enfoque** y ocultar todo menos tu texto.
Consulta [Vistas y modos](views-and-modes).
