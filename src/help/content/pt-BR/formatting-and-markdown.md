# Formatação e markdown

Formate texto de três maneiras: a barra de ferramentas flutuante, atalhos de teclado
ou abreviações markdown digitadas enquanto escreve.

## Resumo rápido

- Digite `## ` para um título, `- ` para um marcador, `1. ` para uma lista numerada.
- Envolva uma palavra com `**negrito**` ou `*itálico*`.
- Selecione texto para usar a barra de ferramentas flutuante.

## Atalhos markdown

Digite no início de uma linha (ou ao redor de uma palavra) e eles se transformam automaticamente:

- `# ` → título 1, `## ` → título 2, `### ` → título 3
- `- ` ou `* ` → lista com marcadores
- `1. ` → lista numerada
- `> ` → citação em bloco
- `**negrito**` → **negrito**, `*itálico*` → *itálico*

## Formatação inline

Selecione o texto e use a barra de ferramentas flutuante, ou use o teclado:

- **Negrito** — o atalho de negrito
- *Itálico* — o atalho de itálico

## Títulos e estrutura

Os títulos definem a **estrutura** do documento exibida no inspetor. Use-os para manter
textos longos navegáveis.

## Listas e citações

Listas e citações em bloco mantêm a formatação ao pressionar Enter, então você pode
continuar digitando sem precisar do mouse. Pressione Enter num item de lista vazio
para sair da lista.
