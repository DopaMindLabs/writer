# Histórico de versões

Cada documento guarda seu próprio histórico para que você possa ver como um rascunho
evoluiu e voltar a uma versão anterior quando precisar. Assim como tudo o mais, esse
histórico é armazenado localmente no navegador.

## Como as versões são salvas

- **Linha de base.** Na primeira vez que você abre um documento, o estado atual é
  registrado como ponto de partida.
- **Automático.** Enquanto você escreve, o documento é salvo em segundo plano como
  instantâneo a cada poucos minutos, mas só quando o texto tiver sido alterado.
- **Manual.** Salve uma versão por conta própria a qualquer momento na aba **Histórico**
  ou **Ações** do inspetor de documento. Dê um nome (por exemplo "antes da revisão")
  para identificá-la facilmente depois.

As versões automáticas são podadas com o tempo para manter o armazenamento sob
controle. Versões que você **fixa** e versões salvas **manualmente** nunca são removidas
automaticamente.

## Visualizando e comparando versões

Abra o inspetor de documento e escolha a aba **Histórico** para ver a lista, da mais
recente. Selecione **histórico completo de versões** para abrir a visualização de
comparação, onde você pode escolher uma versão anterior e ver como ela difere do texto
atual. Alterne entre o diff **inline** e o layout **lado a lado** conforme sua
preferência — texto adicionado e removido são codificados por cores.

## Fixando uma versão

Fixe uma versão para protegê-la da limpeza automática — útil para marcar um marco
como um primeiro rascunho concluído. Fixe e desafixe pela lista de versões.

## Restaurando uma versão anterior

Escolha **Restaurar** em uma versão para reverter o documento a ela. Antes de o
documento ser sobrescrito, o texto atual é salvo automaticamente como uma nova versão,
então uma restauração pode sempre ser desfeita.
