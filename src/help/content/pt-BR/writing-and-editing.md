# Escrita e edição

O editor é construído para prosa de formato longo.

## Resumo rápido

- **Digite** — tudo é salvo automaticamente no navegador instantaneamente.
- **Selecione texto** — uma barra de ferramentas flutuante aparece para formatação rápida.
- **Renomeie** — clique no nome do documento na barra superior.
- **Foco** — pressione **⌘\** para ocultar tudo exceto suas palavras.

## Salvamento automático

Não há botão de salvar e nenhum risco de perder um rascunho. Como o armazenamento
é **local**, sua escrita nunca sai do dispositivo a não ser que você exporte —
veja [Seus dados](your-data).

## A barra de ferramentas flutuante

Selecione qualquer texto para revelar uma pequena barra de ferramentas flutuante com
as formatações mais comuns. Prefere o teclado? Cada ação também tem um atalho —
veja [Formatação e markdown](formatting-and-markdown).

Você pode desativar a barra de ferramentas flutuante nas
[Configurações rápidas](customization-and-settings#quick-settings).

## Renomeando um documento

Clique no nome do documento na barra superior para renomeá-lo. No celular, toque no
botão **⋮** na linha do documento na gaveta de navegação e escolha **Renomear**. Se
você nunca der um nome ao documento, um título é inferido da primeira linha para que
a barra lateral permaneça legível.

## Mantendo o foco

Mude para o **modo foco** (**⌘\**) para ocultar tudo exceto o seu texto. Veja
[Visualizações e modos](views-and-modes).
