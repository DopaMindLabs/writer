# BrainSpace

Um canvas para capturar notas soltas, ideias e fragmentos antes de virarem prosa.
Cada espaço tem o seu.

## Resumo rápido

- **Adicione uma nota** pela barra de ferramentas; abra-a na **gaveta de detalhes**
  para editar.
- Edite o **título** ou o **corpo** de uma nota diretamente no canvas.
- Dê um **tipo** às notas para distinguir ideias, perguntas e fontes.
- Desenhe **conexões** entre notas e **vincule uma nota a um documento**.
- Anexe até **duas imagens** por nota.

## Notas

Adicione notas livremente ao canvas. Uma nota pode conter um pensamento rápido, uma
citação ou uma tarefa. Novas notas aparecem na **visualização atual** — mesmo quando
você rolou para longe do ponto de partida — então o cartão que você adiciona fica
sempre onde você está olhando. Clique no título ou no corpo de uma nota para
**editar diretamente**, ou abra a **gaveta de detalhes** para a visualização completa.
Clique com o botão direito em uma nota para o menu de contexto; **Escape** o fecha.
A barra lateral mostra a **contagem de notas não classificadas**.

## Tipos de notas

As notas têm diferentes **tipos** para que você possa distinguir uma ideia de uma
pergunta ou de uma fonte com um olhar — uma estrutura leve num canvas livre.

## Conexões

Desenhe **conexões** entre notas para mapear como as ideias se relacionam.

## Imagens

Anexe imagens a uma nota pela gaveta de detalhes (até **duas por nota**). Elas são
mantidas localmente entre os recarregamentos e podem ser removidas a qualquer momento.

### Visualizar uma imagem em tamanho real

Clique em qualquer imagem — uma miniatura numa nota, na gaveta de detalhes ou a imagem
grande num cartão de imagem — para abri-la em tamanho real. Quando uma nota tem mais
de uma imagem, navegue entre elas com as setas na tela ou com as teclas de seta Esquerda
e Direita do teclado. Pressione **Escape** ou clique fora da imagem para fechá-la.

### Cartões de imagem

Para uma nota centrada em imagem, adicione um cartão de **imagem** pela barra de
ferramentas do canvas. Um cartão de imagem exibe a imagem em tamanho grande no lugar
do texto do corpo, com o título como legenda opcional. Um novo cartão de imagem começa
com uma zona de soltar **adicionar uma imagem** — clique para escolher sua imagem.
Cartões de imagem suportam até duas imagens e se conectam a outras notas como qualquer
outro cartão.

## Trabalhando ao lado do rascunho

Use a [Visualização dividida](views-and-modes#split) para manter o BrainSpace ao lado
do documento e transferir ideias capturadas para o rascunho enquanto escreve.
