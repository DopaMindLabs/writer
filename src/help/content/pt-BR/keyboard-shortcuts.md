# Atalhos de teclado

Mantenha as mãos no teclado. No macOS use **⌘**; no Windows e Linux use **Ctrl**.

## Resumo rápido

- **⌘K** — abra a Ajuda rápida de qualquer lugar.
- **⌘\** — alterne o modo foco.
- **Esc** — feche uma sobreposição ou cancele uma edição.
- **Ctrl/⌘ + Enter** — salve uma citação que está editando.

## Universais

- **⌘K** — abra a sobreposição de **Ajuda rápida** de qualquer lugar. Pesquise na
  Help Center ou vá para um atalho, depois pressione **Esc** para fechar.
- **⌘\** — alterne o [modo foco](views-and-modes#focus) ligando e desligando.

## Markdown no editor

Digitados no início de uma linha, eles se transformam enquanto você escreve:

- `# ` / `## ` / `### ` — títulos
- `- ` / `* ` — lista com marcadores
- `1. ` — lista numerada
- `> ` — citação em bloco

Envolva uma palavra com `**` para negrito ou `*` para itálico.

## Painel de citações

- **Esc** — feche o painel (ou cancele uma edição em andamento)
- **Ctrl/⌘ + Enter** — salve a entrada que está editando

## Uma nota sobre atalhos do navegador

Algumas combinações de teclas são reservadas pelo navegador ou sistema operacional.
Onde o aplicativo define um atalho, ele tem precedência nas próprias telas; em outros
lugares, o padrão do sistema prevalece.
