# Formázás és markdown

Háromféleképpen formázhat szöveget: az úszó eszközsorral, billentyűparancsokkal
vagy az írás közben begépelt markdown gyorsbillentyűkkel.

## Áttekintés

- Írjon `## ` fejlécet, `- ` felsorolásjelöt, `1. ` számozott listát.
- Zárjon egy szót `**félkövér**` vagy `*dőlt*` jelölésbe.
- Jelöljön ki szöveget az úszó eszközsor használatához.

## Markdown gyorsbillentyűk

Írja ezeket egy sor elejére (vagy egy szó köré), és automatikusan átalakul:

- `# ` → 1. fejlécszint, `## ` → 2. fejlécszint, `### ` → 3. fejlécszint
- `- ` vagy `* ` → felsorolásos lista
- `1. ` → számozott lista
- `> ` → blokk-idézet
- `**félkövér**` → **félkövér**, `*dőlt*` → *dőlt*

## Inline formázás

Jelöljön ki szöveget, és használja az úszó eszközpanelt, vagy a billentyűzetet:

- **Félkövér** — félkövér billentyűparancs
- *Dőlt* — dőlt billentyűparancs

## Fejlécek és szerkezet

A fejlécek meghatározzák a dokumentum **vázlatát**, amelyet a vizsgáló mutat.
Használja ezeket, hogy a hosszú szövegek navigálhatók maradjanak.

## Listák és idézetek

A listák és blokkidézetek megtartják formázásukat Enter lenyomásakor, így
folytathatja a gépelést anélkül, hogy az egérhez nyúlna. Nyomjon Enter üres
listaelemen a lista elhagyásához.
