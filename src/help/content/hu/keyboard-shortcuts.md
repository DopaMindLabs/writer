# Billentyűparancsok

Tartsa a kezeit a billentyűzeten. macOS-en használja a **⌘** billentyűt;
Windows és Linux rendszeren a **Ctrl** billentyűt.

## Áttekintés

- **⌘K** — a Gyors súgó megnyitása bárhonnan.
- **⌘\** — fókusz mód be/kikapcsolása.
- **Esc** — felugró ablak bezárása vagy szerkesztés visszavonása.
- **Ctrl/⌘ + Enter** — a szerkesztett hivatkozás mentése.

## Általános

- **⌘K** — a **Gyors súgó** felugró ablak megnyitása bárhonnan. Keressen a Help Center-ben
  vagy ugorjon egy billentyűparancsra, majd nyomja meg az **Esc** billentyűt a bezáráshoz.
- **⌘\** — a [fókusz mód](views-and-modes#focus) be- és kikapcsolása.

## Szerkesztői markdown

Egy sor elejére gépelve az írás közben ezek alakulnak:

- `# ` / `## ` / `### ` — fejlécek
- `- ` / `* ` — felsorolásos lista
- `1. ` — számozott lista
- `> ` — blokk-idézet

Zárjon be egy szót `**` jelekkel a félkövérhez vagy `*` jelekkel a dőlthöz.

## Hivatkozások panel

- **Esc** — a panel bezárása (vagy folyamatban lévő szerkesztés visszavonása)
- **Ctrl/⌘ + Enter** — a szerkesztett bejegyzés mentése

## Megjegyzés a böngészőbillentyűkről

Néhány billentyűkombinációt a böngésző vagy az operációs rendszer foglal le. Ahol
az alkalmazás billentyűparancsot határoz meg, az a saját képernyőin elsőbbséget
élvez; máshol a rendszer alapértelmezése érvényesül.
