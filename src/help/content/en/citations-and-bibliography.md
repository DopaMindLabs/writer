# Citations & bibliography

Each space has its own citations library for managing references, with full
**BibTeX** import and export.

## At a glance

- **Add** a reference manually, or **import** a `.bib` file / pasted BibTeX.
- **Search**, **edit**, and **delete** entries; duplicates are skipped on import.
- **Bulk-select** to set a type or delete many at once.
- **Export** the whole library back to `.bib`.

## Importing references

Paste or import BibTeX and the entries are added to the space's library. The
parser reads standard BibTeX fields and **diverse entry types**, and reports how
many were added versus skipped as **duplicates**.

## Managing the library

- Browse and **search** entries in the citations pane; a **clear** button
  restores the full list.
- Open an entry for a **detail view**, toggle **edit**, and save inline.
- **Bulk-select** per row or select-all to set a type or delete via a
  confirmation dialogue.

## Exporting

Export the library back to BibTeX at any time to use it elsewhere or to keep a
backup alongside your prose. See [Your data](your-data) for full export options.

## Keyboard

Inside the citations pane:

- **Esc** — close the pane (or cancel an in-progress edit).
- **Ctrl/⌘ + Enter** — save the entry you're editing.
