# Citations & bibliography

Each space has a citations library for managing references, with full **BibTeX**
import and export.

## Importing references

Paste or import BibTeX and the entries are added to the space's library. The
parser reads standard BibTeX fields, so references exported from your reference
manager come across cleanly.

## Managing the library

- Browse and search entries in the citations pane.
- Edit an entry's fields inline; changes save automatically.
- Remove entries you no longer need.

## Exporting

Export the library back to BibTeX at any time to use it elsewhere or to keep a
backup alongside your prose. See [Your data](your-data) for full export options.

## Tips

- Keep one library per space so references stay scoped to the project.
- Pair citations with the [BrainSpace](brainspace) canvas to connect sources to
  the ideas they support.

## Keyboard

Inside the citations pane, **Esc** closes it and **Ctrl/⌘ + Enter** saves the
entry you are editing.
