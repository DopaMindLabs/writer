# Keyboard shortcuts

Keep your hands on the keyboard. On macOS use **⌘**; on Windows and Linux use
**Ctrl**.

## Global

- **⌘K** — open the Quick Help overlay from anywhere. Search the Help Center or
  jump to a shortcut, then press **Esc** to close.
- **⌘\\** — toggle focus mode on and off.

## Editor markdown

Typed at the start of a line, these transform as you write:

- `# ` / `## ` / `### ` — headings
- `- ` / `* ` — bulleted list
- `1. ` — numbered list
- `> ` — block quote

Wrap a word with `**` for bold or `*` for italic.

## Citations pane

- **Esc** — close the pane
- **Ctrl/⌘ + Enter** — save the entry you are editing

## A note on browser shortcuts

A few key combinations are reserved by your browser or operating system. Where
the app defines a shortcut, it takes precedence on its own screens; elsewhere
the system default wins.
