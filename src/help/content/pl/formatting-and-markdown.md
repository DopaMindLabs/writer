# Formatowanie i markdown

Formatuj tekst na trzy sposoby: pływający pasek narzędzi, skróty klawiszowe lub
skróty markdown wpisywane podczas pisania.

## W skrócie

- Wpisz `## ` dla nagłówka, `- ` dla punktu listy, `1. ` dla listy numerowanej.
- Otocz słowo znakami `**pogrubienie**` lub `*kursywa*`.
- Zaznacz tekst, aby użyć zamiast tego pływającego paska narzędzi.

## Skróty markdown

Wpisane na początku wiersza (lub wokół słowa) przekształcają się automatycznie:

- `# ` → nagłówek 1, `## ` → nagłówek 2, `### ` → nagłówek 3
- `- ` lub `* ` → lista punktowana
- `1. ` → lista numerowana
- `> ` → cytat blokowy
- `**pogrubienie**` → **pogrubienie**, `*kursywa*` → *kursywa*

## Formatowanie inline

Zaznacz tekst i użyj pływającego paska narzędzi lub klawiatury:

- **Pogrubienie** — skrót pogrubienia
- *Kursywa* — skrót kursywy

## Nagłówki i struktura

Nagłówki definiują **konspekt** dokumentu widoczny w inspektorze. Używaj ich, aby
długie teksty były nawigowalne.

## Listy i cytaty

Listy i cytaty blokowe zachowują formatowanie po naciśnięciu Enter, więc możesz
pisać dalej bez sięgania po mysz. Naciśnij Enter na pustym elemencie listy, aby
opuścić listę.
