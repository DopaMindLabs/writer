# Skróty klawiszowe

Trzymaj ręce na klawiaturze. Na macOS używaj **⌘**; na Windows i Linux używaj **Ctrl**.

## W skrócie

- **⌘K** — otwórz Szybką pomoc z dowolnego miejsca.
- **⌘\** — przełącz tryb skupienia.
- **Esc** — zamknij nakładkę lub anuluj edycję.
- **Ctrl/⌘ + Enter** — zapisz edytowany cytat.

## Globalne

- **⌘K** — otwórz nakładkę **Szybka pomoc** z dowolnego miejsca. Przeszukuj
  Help Center lub przejdź do skrótu, a następnie naciśnij **Esc**, aby zamknąć.
- **⌘\** — przełącz [tryb skupienia](views-and-modes#focus) włącz/wyłącz.

## Markdown w edytorze

Wpisane na początku wiersza przekształcają się podczas pisania:

- `# ` / `## ` / `### ` — nagłówki
- `- ` / `* ` — lista punktowana
- `1. ` — lista numerowana
- `> ` — cytat blokowy

Otocz słowo znakami `**` dla pogrubienia lub `*` dla kursywy.

## Panel cytatów

- **Esc** — zamknij panel (lub anuluj trwającą edycję)
- **Ctrl/⌘ + Enter** — zapisz edytowany wpis

## Uwaga dotycząca skrótów przeglądarki

Niektóre kombinacje klawiszy są zarezerwowane przez przeglądarkę lub system operacyjny.
Gdzie aplikacja definiuje skrót, ma on pierwszeństwo na własnych ekranach; w innych
miejscach wygrywa domyślne zachowanie systemu.
