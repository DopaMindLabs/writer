# Formátování a markdown

Formátujte text třemi způsoby: plovoucím panelem nástrojů, klávesovými zkratkami
nebo zkratkami markdownu zadávanými při psaní.

## Přehled

- Napište `## ` pro nadpis, `- ` pro odrážku, `1. ` pro číslovaný seznam.
- Obklopte slovo `**tučně**` nebo `*kurzívou*`.
- Vyberte text a použijte plovoucí panel nástrojů.

## Zkratky markdownu

Napište tato na začátek řádku (nebo kolem slova) a automaticky se transformují:

- `# ` → nadpis 1, `## ` → nadpis 2, `### ` → nadpis 3
- `- ` nebo `* ` → seznam s odrážkami
- `1. ` → číslovaný seznam
- `> ` → blokový citát
- `**tučně**` → **tučně**, `*kurzíva*` → *kurzíva*

## Vložené formátování

Vyberte text a použijte plovoucí panel nástrojů, nebo použijte klávesnici:

- **Tučné** — zkratka pro tučné
- *Kurzíva* — zkratka pro kurzívu

## Nadpisy a struktura

Nadpisy definují **osnovu** dokumentu zobrazenou v inspektoru. Používejte je,
aby dlouhé texty zůstaly přehledné.

## Seznamy a citáty

Seznamy a blokové citáty si zachovají formátování po stisknutí Enter, takže
můžete pokračovat v psaní bez sáhnutí po myši. Stiskněte Enter na prázdné
položce seznamu pro ukončení seznamu.
