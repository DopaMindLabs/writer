# හැඩතල ගැන්වීම සහ Markdown

පෙළ හැඩතල ගැන්වීමේ ක්‍රම තුනකි: පාවෙන මෙවලම් තීරුව, යතුරුපුවරු කෙටිමං, හෝ ලිවීමේදී ටයිප් කළ markdown කෙටිමං.

## දළ විශ්ලේෂණය

- ශීර්ෂයක් සඳහා `## ` ටයිප් කරන්න, ලස්සනක් සඳහා `- `, අංකිත ලැයිස්තුවක් සඳහා `1. `.
- වචනයකට `**bold**` හෝ `*italic*` ලෙස ලනෙනු ලාන්න.
- ඒ වෙනුවට පාවෙන මෙවලම් තීරුව භාවිත කිරීමට පෙළ තෝරන්න.

## Markdown කෙටිමං

පේළියක ආරම්භයේ (හෝ වචනයකට වටා) ටයිප් කිරීමෙන් ස්වයංක්‍රීයව පරිවර්තනය:

- `# ` → ශීර්ෂ 1, `## ` → ශීර්ෂ 2, `### ` → ශීර්ෂ 3
- `- ` හෝ `* ` → ලස්සනයුත් ලැයිස්තුව
- `1. ` → අංකිත ලැයිස්තුව
- `> ` → ඒකාබද්ධ උද්ධෘතය
- `**bold**` → **bold**, `*italic*` → *italic*

## ඉනිල හැඩතල ගැන්වීම

පෙළ තෝරා පාවෙන මෙවලම් තීරුව භාවිත කරන්න, හෝ යතුරුපුවරු භාවිත කරන්න:

- **Bold** — bold කෙටිය
- *Italic* — italic කෙටිය

## ශීර්ෂ සහ ව්‍යූහය

ශීර්ෂ පරීක්ෂකයේ දිස්වන ලේඛනයේ **ගෙනල** නිර්ණය කරයි. දිගු කෑලි ගමන් කළ හැකිව තබා ගැනීමට ඒවා භාවිත කරන්න.

## ලැයිස්තු සහ උද්ධෘත

ලැයිස්තු සහ ඒකාබද්ධ උද්ධෘත Enter ලාද විට ඒවායේ හැඩතල ගැන්වීම රැකෙයි, ඒ නිසා මකෙකු නොයොදා ලිවීම දිගටම කළ හැකිය. හිස් ලැයිස්තු අයිතමයකට Enter ලාද ලැයිස්තුවෙන් ඉවත් වේ.
