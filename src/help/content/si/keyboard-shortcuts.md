# යතුරුපුවරු කෙටිමං

යතුරුපුවරුව මත අත් තබාගන්න. macOS හි **⌘** භාවිතා කරන්න; Windows සහ Linux හි **Ctrl** භාවිතා කරන්න.

## දළ විශ්ලේෂණය

- **⌘K** — ඕනෑ තැනකදී ඉක්මන් උදවු විවෘත කරන්න.
- **⌘\\** — අවධාන ආකාරය ස්ථිතිමත් කරන්න.
- **Esc** — ආවරණයක් වැසීම හෝ සංස්කරණයක් අවලංගු කිරීම.
- **Ctrl/⌘ + Enter** — ඔබ සංස්කරණය කරන උද්ධෘතය සුරකිනු ලැබේ.

## සාර්වත්‍රික

- **⌘K** — ඕනෑ තැනකදී **ඉක්මන් උදවු** ආවරණය විවෘත කරන්න. උදවු මධ්‍යස්ථානය සොයන්න හෝ කෙටිමඟකට පනිනු ලැබේ, ඉන්පසු **Esc** 누ඇ: ද.
- **⌘\\** — [අවධාන ආකාරය](views-and-modes#focus) ක්‍රියාත්මක/අක්‍රිය කරන්න.

## සංස්කරක Markdown

පේළියේ ආරම්භයේ ටයිප් කළ විට, ලිවීමේදී රූපාන්තරනය වේ:

- `# ` / `## ` / `### ` — ශීර්ෂ
- `- ` / `* ` — ලිපිය ලැයිස්තුව
- `1. ` — අංකිත ලැයිස්තුව
- `> ` — කොටු උද්ධෘතය

**bold** සඳහා `**` හෝ *italic* සඳහා `*` ෙව් ෙ වචනයක් ඔතා ගන්න.

## උද්ධෘත කවුළුව

- **Esc** — කවුළුව වසන්න (හෝ ක්‍රියාත්මක සංස්කරණයක් අවලංගු කරන්න)
- **Ctrl/⌘ + Enter** — ඔබ සංස්කරණය කරන ඇතුළත් කිරීම සුරකිනු ලැබේ

## බ්‍රවුසර කෙටිමං ගැන සටහනක්

ඔබේ බ්‍රවුසරය හෝ මෙහෙයුම් පද්ධතිය යතුරු කිහිපයක් රාඛා ගනී. යෙදුම කෙටිමඟක් නිර්ණය කරන විට, එය තමාගේ තිර මත ප්‍රමුඛතාවය ගනී; වෙනත් තැන් ෙව්ෙ පද්ධතිය ජය ගනී.
