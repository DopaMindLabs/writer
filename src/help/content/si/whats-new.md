# අලුත් දේ

LIpsum Writer ෙ නවතම දියුණුකම්, නවතම ෙ. සම්පූර්ණ change log සඳහා, GitHub ෙ ව්‍යාපෘතියේ නිකුතු සටහන් බලන්න.

## ජූනි 2026

- **දුරකථනවල Split view** — ෙ ෙ දෙකක් දැන් portrait ෙ ශ්‍රේණිගත ව, landscape ෙ සාථ-සසඳා. [ජංගම](mobile) බලන්න.
- **ජංගමෙ ලේඛන පරීක්ෂකය** — **තවත්** ටැබ් ෙ ගෙනල, තොරතුරු, ඉතිහාසය, සහ ක්‍රියා විවෘත කරන්න. [ලේඛන පරීක්ෂකය](doc-inspector) බලන්න.
- **සංචාර skip හඳුනාගනී** — සංචාරය අතරමඟ ඉවත් වී, සංචාර වෙනත් තිරවල ස්වයංක්‍රීයව ආරම්භ නොවේ. ඕනෑ සංචාරයක් උදව් (**?**) මෙනුවෙ නැවත ක්‍රීඩා කරන්න.
- **Sync තත්ත්වය** — sync සහ backups සක්‍රීය කරන තෙක් නිවාස තිරය අනතුරු chip දක්වයි. [ඔබේ දත්ත](your-data) බලන්න.
- **Contact** — **තවත්** මෙනුවෙ සම්බන්ධ වන්න; එය ව්‍යාපෘතියේ tracker ෙ නව issue ෙ.
