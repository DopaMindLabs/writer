# የቁልፍ ሰሌዳ አቋራጮች

እጆቻችሁ ቁልፍ ሰሌዳ ላይ ይቆዩ። macOS ላይ **⌘** ተጠቀሙ፤ Windows እና Linux ላይ **Ctrl**።

## በአጭሩ

- **⌘K** — ፈጣን እርዳታ ከየትም ክፈት።
- **⌘\\** — የትኩረት ሁነታ ማብ/ማጥፋ።
- **Esc** — ድርቡን ዝጋ ወይም አርትዖት ሰርዝ።
- **Ctrl/⌘ + Enter** — እያስተካከልክ ያለህ ጥቅስ አስቀምጥ።

## ዓለም አቀፋዊ

- **⌘K** — **ፈጣን እርዳታ** ድርቡን ከየትም ክፈት። Help Center ፈልግ ወይም ወደ አቋራጭ ዝለል — ለመዝጋት **Esc** ጫን።
- **⌘\\** — [የትኩረት ሁነታ](views-and-modes#focus) ማብ/ማጥፋ።

## የአርታዒ ማርክዳውን

ከመስመር መጀመሪያ ላይ ሲጻፉ ወዲያው ይቀየራሉ፡

- `# ` / `## ` / `### ` — ርዕሶች
- `- ` / `* ` — ምርጫ ዝርዝር
- `1. ` — ቁጥር ዝርዝር
- `> ` — ብሎክ ጥቅስ

ቃልን `**` ሸፍን ደማቅ ለማድረግ ወይም `*` ሰያፍ ለማድረግ።

## የጥቅስ ፓነል

- **Esc** — ፓነሉን ዝጋ (ወይም እያለ ያለ አርትዖት ሰርዝ)
- **Ctrl/⌘ + Enter** — እያስተካከልከው ያለውን ግቤት አስቀምጥ

## ስለ ብሮውዘር አቋራጮች

ብሮውዘርህ ወይም ስርዓቱ ጥቂት ቁልፍ ጥምሮቹን ይይዛሉ። አፕ አቋራጭ ሲያስቀምጥ በራሱ ማያ ፊደሎቹ ቅድሚያ ይሰጣዋል፤ ሌላ ቦታ ስርዓቱ ነባሪ ቅድሚያ ያገኛል።
