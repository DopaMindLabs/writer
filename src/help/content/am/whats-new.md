# አዲስ ምን አለ

LIpsum Writer ውስጥ የቅርብ ጊዜ መሻሻሎቹ — አዲሱ ቀዳሚ። ሙሉ የቀደም ማስታወሻ ለማየት፣ GitHub ላይ ፕሮጀክቱ ሪሊዝ ማስታወሻዎቹ ይመልከቱ።

## ሰኔ 2026

- **ስልክ ላይ ስፕሊት ዕይታ** — ሁለቱ ፓነሎቹ ፖርትሬይት ላይ ከላይ ወደ ታች ይደርደራሉ፣ ላንድስኬፕ ላይ ጎን ለጎን ይቀመጣሉ። [ሞባይል](mobile)ን ይመልከቱ።
- **ሞባይል ላይ Doc inspector** — ንድፍ፣ መረጃ፣ ታሪክ እና እርምጃዎቹ **ተጨማሪ** ትር ይከፈታሉ። [የሰነድ ተቆጣጣሪ](doc-inspector)ን ይመልከቱ።
- **ጉብኝቶቹ ሰርዙ ያስቡ** — ሚመሩ ጉብኝቶቹ አዕምሮ ሳሉ ሰርዛቸው — ሌሎቹ ማያ ፊደሎቹ ላይ ጉብኝቶቹ ራሳቸው አይጀምሩም። ማንኛውም ጉብኝት ከእርዳታ (**?**) ምናሌ ደጋግሙ።
- **ሲንክ ሁኔታ በቅጽበት** — ዋና ገጽ ሲንክ እና ምትኬዎቹ ሳይነቃ ወዲያው ማስጠንቀቂያ ቺፕ ይሳያሉ። [የእርስዎ ውሂብ](your-data)ን ይመልከቱ።
- **ያነጋግሩን** — **ተጨማሪ** ምናሌ ያስደርሱ — ፕሮጀክቱ ትራከር ላይ አዲስ ጉዳይ ይከፈታሉ።
