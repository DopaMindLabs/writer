# Συντομεύσεις πληκτρολογίου

Κρατήστε τα χέρια σας στο πληκτρολόγιο. Στο macOS χρησιμοποιήστε **⌘**· στα
Windows και Linux χρησιμοποιήστε **Ctrl**.

## Σύντομη επισκόπηση

- **⌘K** — ανοίγει τη Γρήγορη βοήθεια από οπουδήποτε.
- **⌘\** — εναλλαγή λειτουργίας εστίασης.
- **Esc** — κλείνει επικάλυψη ή ακυρώνει επεξεργασία.
- **Ctrl/⌘ + Enter** — αποθηκεύει παραπομπή που επεξεργάζεστε.

## Καθολικές

- **⌘K** — ανοίγει την επικάλυψη **Γρήγορη βοήθεια** από οπουδήποτε. Αναζητήστε στο Help Center
  ή μεταβείτε σε συντόμευση, έπειτα πατήστε **Esc** για κλείσιμο.
- **⌘\** — εναλλαγή [λειτουργίας εστίασης](views-and-modes#focus) ενεργή/ανενεργή.

## Markdown επεξεργαστή

Πληκτρολογημένα στην αρχή γραμμής, αυτά μετασχηματίζονται καθώς γράφετε:

- `# ` / `## ` / `### ` — τίτλοι
- `- ` / `* ` — λίστα με κουκκίδες
- `1. ` — αριθμημένη λίστα
- `> ` — αποσπασματικό απόσπασμα

Τυλίξτε λέξη με `**` για έντονο ή `*` για πλάγιο.

## Παράθυρο παραπομπών

- **Esc** — κλείνει το παράθυρο (ή ακυρώνει επεξεργασία σε εξέλιξη)
- **Ctrl/⌘ + Enter** — αποθηκεύει την εγγραφή που επεξεργάζεστε

## Σημείωση για συντομεύσεις προγράμματος περιήγησης

Μερικοί συνδυασμοί πλήκτρων είναι δεσμευμένοι από το πρόγραμμα περιήγησης ή το
λειτουργικό σύστημα. Όπου η εφαρμογή ορίζει συντόμευση, αυτή έχει προτεραιότητα
στις δικές της οθόνες· αλλού κερδίζει η προεπιλογή συστήματος.
