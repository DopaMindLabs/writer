# Μορφοποίηση και markdown

Μορφοποιήστε κείμενο με τρεις τρόπους: την κινητή γραμμή εργαλείων, συντομεύσεις
πληκτρολογίου ή συντομεύσεις markdown που πληκτρολογείτε καθώς γράφετε.

## Σύντομη επισκόπηση

- Πληκτρολογήστε `## ` για τίτλο, `- ` για bullet, `1. ` για αριθμημένη λίστα.
- Τυλίξτε λέξη σε `**έντονο**` ή `*πλάγιο*`.
- Επιλέξτε κείμενο για χρήση της κινητής γραμμής εργαλείων.

## Συντομεύσεις markdown

Πληκτρολογήστε αυτά στην αρχή γραμμής (ή γύρω από λέξη) και μετασχηματίζονται
αυτόματα:

- `# ` → τίτλος 1, `## ` → τίτλος 2, `### ` → τίτλος 3
- `- ` ή `* ` → λίστα με κουκκίδες
- `1. ` → αριθμημένη λίστα
- `> ` → αποσπασματικό απόσπασμα
- `**έντονο**` → **έντονο**, `*πλάγιο*` → *πλάγιο*

## Ενσωματωμένη μορφοποίηση

Επιλέξτε κείμενο και χρησιμοποιήστε την κινητή γραμμή εργαλείων ή το πληκτρολόγιο:

- **Έντονο** — συντόμευση έντονου
- *Πλάγιο* — συντόμευση πλάγιου

## Τίτλοι και δομή

Οι τίτλοι ορίζουν τη **διάρθρωση** εγγράφου που εμφανίζεται στον επιθεωρητή.
Χρησιμοποιήστε τους για να διατηρείτε τα μεγάλα κείμενα πλοηγήσιμα.

## Λίστες και αποσπάσματα

Λίστες και αποσπάσματα μπλοκ διατηρούν τη μορφοποίησή τους με πάτημα Enter,
ώστε να συνεχίζετε να πληκτρολογείτε χωρίς να φτάνετε στο ποντίκι. Πατήστε
Enter σε κενό στοιχείο λίστας για έξοδο από λίστα.
