# Formatação e markdown

Formate texto de três formas: a barra de ferramentas flutuante, atalhos de teclado
ou abreviaturas markdown escritas enquanto escreve.

## Resumo

- Escreva `## ` para um título, `- ` para um marcador, `1. ` para uma lista numerada.
- Envolva uma palavra com `**negrito**` ou `*itálico*`.
- Seleccione texto para usar a barra de ferramentas flutuante.

## Atalhos markdown

Escritos no início de uma linha (ou à volta de uma palavra) e transformam-se automaticamente:

- `# ` → título 1, `## ` → título 2, `### ` → título 3
- `- ` ou `* ` → lista com marcadores
- `1. ` → lista numerada
- `> ` → citação em bloco
- `**negrito**` → **negrito**, `*itálico*` → *itálico*

## Formatação inline

Seleccione texto e use a barra de ferramentas flutuante, ou use o teclado:

- **Negrito** — o atalho de negrito
- *Itálico* — o atalho de itálico

## Títulos e estrutura

Os títulos definem a **estrutura** do documento apresentada no inspector. Use-os para
manter textos longos navegáveis.

## Listas e citações

As listas e citações em bloco mantêm a formatação ao premir Enter, para que possa
continuar a escrever sem precisar do rato. Prima Enter num item de lista vazio para
sair da lista.
