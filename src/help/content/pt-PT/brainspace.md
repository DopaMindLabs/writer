# BrainSpace

Uma tela para capturar notas soltas, ideias e fragmentos antes de se tornarem prosa.
Cada espaço tem a sua.

## Resumo

- **Adicione uma nota** a partir da barra de ferramentas; abra-a na **gaveta de detalhes**
  para editar.
- Edite o **título** ou o **corpo** de uma nota directamente na tela.
- Atribua um **tipo** às notas para distinguir ideias, questões e fontes.
- Desenhe **ligações** entre notas e **ligue uma nota a um documento**.
- Anexe até **duas imagens** por nota.

## Notas

Adicione notas livremente à tela. Uma nota pode conter um pensamento rápido, uma
citação ou uma tarefa. As novas notas aparecem na **vista actual** — mesmo quando se
deslocou para longe do ponto de partida — pelo que um cartão que adiciona está sempre
onde está a olhar. Clique no título ou no corpo de uma nota para **editar directamente**,
ou abra a **gaveta de detalhes** para a vista completa. Clique com o botão direito
numa nota para o menu de contexto; **Escape** fecha-o. A barra lateral mostra a
**contagem de notas não ordenadas**.

## Tipos de notas

As notas têm diferentes **tipos** para que possa distinguir uma ideia de uma questão
ou de uma fonte de relance — uma estrutura leve numa tela de outra forma livre.

## Ligações

Desenhe **ligações** entre notas para mapear como as ideias se relacionam.

## Imagens

Anexe imagens a uma nota a partir da gaveta de detalhes (até **duas por nota**). São
mantidas localmente entre recarregamentos e podem ser removidas em qualquer momento.

### Ver uma imagem em tamanho real

Clique em qualquer imagem — uma miniatura numa nota, na gaveta de detalhes ou a imagem
grande num cartão de imagem — para a abrir em tamanho real. Quando uma nota tem mais
do que uma imagem, navegue entre elas com as setas no ecrã ou com as teclas de seta
Esquerda e Direita do teclado. Prima **Escape** ou clique fora da imagem para a fechar.

### Cartões de imagem

Para uma nota centrada em imagem, adicione um cartão de **imagem** a partir da barra
de ferramentas da tela. Um cartão de imagem mostra a imagem em tamanho grande em vez
do texto do corpo, com o título como legenda opcional. Um novo cartão de imagem começa
com uma zona de largada **adicionar uma imagem** — clique para escolher a sua imagem.
Os cartões de imagem suportam até duas imagens e ligam-se a outras notas como qualquer
outro cartão.

## Trabalhar ao lado do rascunho

Utilize a [Vista dividida](views-and-modes#split) para manter o BrainSpace ao lado
do documento e transferir ideias capturadas para o rascunho enquanto escreve.
