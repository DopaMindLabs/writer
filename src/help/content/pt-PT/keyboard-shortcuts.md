# Atalhos de teclado

Mantenha as mãos no teclado. No macOS use **⌘**; no Windows e Linux use **Ctrl**.

## Resumo

- **⌘K** — abra a Ajuda rápida a partir de qualquer lugar.
- **⌘\** — alterne o modo de foco.
- **Esc** — feche uma sobreposição ou cancele uma edição.
- **Ctrl/⌘ + Enter** — guarde uma citação que está a editar.

## Universais

- **⌘K** — abra a sobreposição de **Ajuda rápida** a partir de qualquer lugar. Pesquise
  no Help Center ou vá para um atalho, depois prima **Esc** para fechar.
- **⌘\** — alterne o [modo de foco](views-and-modes#focus) ligando e desligando.

## Markdown no editor

Escritos no início de uma linha, transformam-se enquanto escreve:

- `# ` / `## ` / `### ` — títulos
- `- ` / `* ` — lista com marcadores
- `1. ` — lista numerada
- `> ` — citação em bloco

Envolva uma palavra com `**` para negrito ou `*` para itálico.

## Painel de citações

- **Esc** — feche o painel (ou cancele uma edição em curso)
- **Ctrl/⌘ + Enter** — guarde a entrada que está a editar

## Uma nota sobre atalhos do navegador

Algumas combinações de teclas são reservadas pelo navegador ou pelo sistema operativo.
Onde a aplicação define um atalho, este tem precedência nos seus próprios ecrãs; noutros
locais, a predefinição do sistema prevalece.
