# Escrita e edição

O editor é construído para prosa de formato longo.

## Resumo

- **Escreva** — tudo é guardado automaticamente no navegador de imediato.
- **Seleccione texto** — aparece uma barra de ferramentas flutuante para formatação rápida.
- **Renomeie** — clique no nome do documento na barra superior.
- **Foco** — prima **⌘\** para ocultar tudo excepto as suas palavras.

## Gravação automática

Não existe botão de guardar nem risco de perder um rascunho. Como o armazenamento é
**local**, a sua escrita nunca sai do dispositivo a não ser que a exporte —
consulte [Os seus dados](your-data).

## A barra de ferramentas flutuante

Seleccione qualquer texto para revelar uma pequena barra de ferramentas flutuante com
a formatação mais comum. Prefere o teclado? Cada acção tem também um atalho —
consulte [Formatação e markdown](formatting-and-markdown).

Pode desactivar a barra de ferramentas flutuante nas
[Definições rápidas](customization-and-settings#quick-settings).

## Mudar o nome de um documento

Clique no nome do documento na barra superior para o renomear. No telemóvel, toque
no botão **⋮** na linha do documento na gaveta de navegação e escolha **Renomear**.
Se nunca der um nome ao documento, um título é inferido da primeira linha para que
a barra lateral se mantenha legível.

## Manter o foco

Mude para o **modo de foco** (**⌘\**) para ocultar tudo excepto o seu texto. Consulte
[Vistas e modos](views-and-modes).
