# Formatēšana un markdown

Formatējiet tekstu trīs veidos: peldošā rīkjosla, tastatūras īsceļi vai markdown
saīsnes, rakstot.

## Kopsavilkums

- Ierakstiet `## ` virsrakstam, `- ` punktu sarakstam, `1. ` numurētam sarakstam.
- Ietiniet vārdu `**treknrakstā**` vai `*slīprakstā*`.
- Iezīmējiet tekstu, lai izmantotu peldošo rīkjoslu.

## Markdown saīsnes

Ierakstiet tās rindas sākumā (vai ap vārdu) un tās automātiski pārveidojas:

- `# ` → 1. līmeņa virsraksts, `## ` → 2. līmeņa virsraksts, `### ` → 3. līmeņa virsraksts
- `- ` vai `* ` → punktu saraksts
- `1. ` → numurēts saraksts
- `> ` → bloka citāts
- `**treknraksts**` → **treknraksts**, `*slīpraksts*` → *slīpraksts*

## Tiešsaites formatēšana

Iezīmējiet tekstu un izmantojiet peldošo rīkjoslu vai tastatūru:

- **Treknraksts** — treknraksta saīsne
- *Slīpraksts* — slīpraksta saīsne

## Virsraksti un struktūra

Virsraksti definē dokumenta **struktūru**, kas parādīta inspektorā. Izmantojiet tos,
lai gari darbi būtu viegli pārvietojami.

## Saraksti un citāti

Saraksti un bloka citāti saglabā formatēšanu, nospiežot Enter, tāpēc varat turpināt
rakstīt, nesniedzot pēc peles. Nospiediet Enter uz tukša saraksta elementa, lai izietu
no saraksta.
