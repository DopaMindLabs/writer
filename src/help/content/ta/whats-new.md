# புதியது என்ன

LIpsum Writer க்கான சமீபத்திய மேம்பாடுகள், புதியவை முதலில். முழு மாற்ற பதிவுக்கு, GitHub இல் திட்டத்தின் வெளியீட்டு குறிப்புகளை பார்க்கவும்.

## ஜூன் 2026

- **தொலைபேசிகளில் பிரிந்த காட்சி** — இரண்டு பலகங்கள் இப்போது நிலை நிறுவனத்தில் மேலும் கீழும் அடுக்கிக்கொண்டு நிலப்பரப்பில் பக்கத்தில் பக்கம் அமர்கின்றன. [மொபைல்](mobile) பாருங்கள்.
- **மொபைலில் ஆவண ஆய்வாளர்** — **மேலும்** தாவலில் இருந்து அவுட்லைன், தகவல், வரலாறு, மற்றும் செயல்களை திறக்கவும். [ஆவண ஆய்வாளர்](doc-inspector) பாருங்கள்.
- **சுற்றுலாக்கள் தவிர்ப்பை மதிக்கின்றன** — வழிகாட்டிய சுற்றுலாவை பாதியில் நிராகரிக்கவும், மற்ற திரைகளில் சுற்றுலாக்கள் தானாக தொடங்குவதை நிறுத்துகின்றன. உதவி (**?**) மெனுவில் இருந்து எந்த சுற்றுலாவையும் மீண்டும் இயக்கவும்.
- **ஒரு பார்வையில் ஒத்திசைவு நிலை** — ஒத்திசைவு மற்றும் காப்புப்பிரதிகள் இயக்கப்படும் வரை முகப்பு திரை எச்சரிக்கை சிப் காட்டுகிறது. [உங்கள் தரவு](your-data) பாருங்கள்.
- **தொடர்பு** — **மேலும்** மெனுவில் இருந்து எங்களை தொடர்பு கொள்ளுங்கள்; அது திட்டத்தின் கண்காணிப்பில் புதிய சிக்கல் திறக்கிறது.
