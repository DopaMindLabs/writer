# Comenzi rapide de tastatură

Păstrați mâinile pe tastatură. Pe macOS folosiți **⌘**; pe Windows și Linux
folosiți **Ctrl**.

## Rezumat

- **⌘K** — deschide Ajutorul rapid de oriunde.
- **⌘\** — comută modul focus.
- **Esc** — închide o suprapunere sau anulează o editare.
- **Ctrl/⌘ + Enter** — salvează o citare pe care o editați.

## Universal

- **⌘K** — deschide fereastra pop-up **Ajutor rapid** de oriunde. Căutați în Help Center
  sau săriți la o comandă rapidă, apoi apăsați **Esc** pentru a închide.
- **⌘\** — activează/dezactivează [modul focus](views-and-modes#focus).

## Markdown în editor

Tastate la începutul unui rând, acestea se transformă pe măsură ce scrieți:

- `# ` / `## ` / `### ` — titluri
- `- ` / `* ` — listă cu marcatori
- `1. ` — listă numerotată
- `> ` — citat bloc

Înconjoară un cuvânt cu `**` pentru bold sau `*` pentru italic.

## Panoul de citări

- **Esc** — închide panoul (sau anulează o editare în curs)
- **Ctrl/⌘ + Enter** — salvează intrarea pe care o editați

## O notă despre comenzile rapide ale browserului

Câteva combinații de taste sunt rezervate de browser sau sistemul de operare. Acolo
unde aplicația definește o comandă rapidă, aceasta are prioritate pe propriile
ecrane; în altă parte, sistemul implicit câștigă.
