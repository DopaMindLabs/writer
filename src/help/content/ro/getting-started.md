# Noțiuni de bază

Un loc de scriere cu date locale. Munca dvs. se află în **spații** și se salvează
automat în browser. Fără conturi, fără cloud, fără buton de salvare.

## Rezumat

1. Din **Acasă**, deschideți sau creați un spațiu.
2. Adăugați un document din **bara laterală**.
3. Începeți să tastați — fiecare apăsare de tastă este salvată.

Acesta este întregul flux. Tot ce urmează sunt detalii la care puteți reveni.

## Primul dvs. document

- Un document este pur și simplu un loc în care scrieți — nu există un dialog „fișier nou".
- Redenumiți-l oricând făcând clic pe numele său din **bara de sus**.
- Nu l-ați numit? Titlul este dedus din primul rând.

## Orientarea în aplicație

- **Bara laterală** — spațiile, secțiunile și documentele dvs.
- **Bara de sus** — documentul curent și filele de mod.
- Link-ul **Ajutor** din navigarea paginii — vă aduce înapoi oricând.

## Tururi ghidate

Prima dată când deschideți un ecran important, un scurt tur interactiv evidențiază
elementele esențiale. Săriți un tur la jumătate și tururile nu mai pornesc automat
pe alte ecrane. Reluați orice tur oricând din meniul de ajutor (**?**) sau din
[Setări rapide](customization-and-settings#quick-settings).
