# Àwọn keyboard shortcut

Pa àwọn ọwọ́ rẹ sílẹ̀ lórí keyboard. Lórí macOS lo **⌘**; lórí Windows àti Linux lo **Ctrl**.

## Ìsọnísókí

- **⌘K** — ṣí Quick Help láti ibikíbi.
- **⌘\\** — yípadà ìṣọkan idojúkọ.
- **Esc** — pa àwòrán tó ṣòfò tàbí fagilé ìṣàtunṣe.
- **Ctrl/⌘ + Enter** — fipamọ́ ìtọ́kasí tí o ń ṣàtunṣe.

## Àgbáyé

- **⌘K** — ṣí àwòrán **Quick Help** láti ibikíbi. Wá nínú Help Center tàbí fo sí kúlẹ̀kúlẹ̀ kan, lẹ́yìn náà tẹ **Esc** láti pa.
- **⌘\\** — yípadà [ìṣọkan idojúkọ](views-and-modes#focus) sẹ́ àti wọ.

## Markdown olùṣàtúpò

Tí a tẹ ní ìbẹ̀rẹ̀ ìlà kan, ìwọ̀nyí yípadà bí o ṣe ń kọ:

- `# ` / `## ` / `### ` — àwọn àkọlé
- `- ` / `* ` — àtòjọ pẹ̀lú ọdẹ lẹ́tà
- `1. ` — àtòjọ tí a nọ́mbà
- `> ` — ìtúmọ̀ àkójọpọ̀

Dì ọ̀rọ̀ kan pọ̀ pẹ̀lú `**` fún bold tàbí `*` fún italic.

## Àpèsè ìtọ́kasí

- **Esc** — pa àpèsè (tàbí fagilé ìṣàtunṣe tí ń lọ lọwọlọwọ)
- **Ctrl/⌘ + Enter** — fipamọ́ ìfàwọlé tí o ń ṣàtunṣe

## Àkọsílẹ̀ nípa àwọn kúlẹ̀kúlẹ̀ aṣàwákiri

Àwọn àkojọpọ̀ bọ́tìnnì díẹ̀ ni aṣàwákiri rẹ tàbí ẹ̀rọ iṣiṣẹ ń pa wọ́n mọ́. Níbi tí ètò bá ṣèdálẹ̀ kúlẹ̀kúlẹ̀, ó ń gbéga lórí àwọn ojú ìwòye rẹ̀; níbi mìíràn, ìpinnu ẹ̀rọ ń ṣe.
