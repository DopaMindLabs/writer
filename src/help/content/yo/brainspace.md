# BrainSpace

Àyè fún gbígba àwọn àkọsílẹ̀ tó ṣìnkú, àwọn ìmọ̀, àti àwọn ẹsẹ̀ kí wọ́n tó di ọrọ tó péye. Aaye kọ̀ọ̀kan ní tirẹ̀.

## Ìsọnísókí

- **Fi àkọsílẹ̀ kun** láti toolbar; ṣí rẹ̀ nínú **ìpàpọ̀ àlàyé** láti ṣàtunṣe.
- Ṣàtunṣe **àkọlé** tàbí **ara** àkọsílẹ̀ taara, ní ibi tí o wà lára canvas.
- Fún àwọn àkọsílẹ̀ ní **iru** láti ṣe ìyàtọ̀ laarin àwọn ìmọ̀, àwọn ìbéèrè, àti àwọn orísun.
- Ya **àwọn ìsopọ̀** laarin àwọn àkọsílẹ̀, tí o sì **so àkọsílẹ̀ mọ́ ìwé kan**.
- Fi **àwọn awòrán méjì** pọ̀ mọ́ àkọsílẹ̀ kọ̀ọ̀kan.

## Àwọn àkọsílẹ̀

Fi àwọn àkọsílẹ̀ kun lórí canvas larọwọto. Àkọsílẹ̀ kan lè gba ìrònú kíákíá, agbasọ, tàbí iṣẹ́ àmúṣẹ. Àwọn àkọsílẹ̀ tuntun farahàn nínú **ìwòye lọwọlọwọ rẹ** — paapaa nígbà tí o bá ti fà jìnnà kúrò nínú ibi tí o bẹ̀rẹ̀ — nítorí náà kọọpu tí o fi kun wà ní ọwọ́ rẹ nígbogbo ìgbà. Tẹ àkọlé tàbí ara àkọsílẹ̀ láti **ṣàtunṣe taara**, tàbí ṣí **ìpàpọ̀ àlàyé** fún ìwòye kíkún. Tẹ ẹ̀tún àkọsílẹ̀ fún àkójọ àṣàyàn rẹ̀; **Escape** pa à mọ́. Ìgbẹ̀yìn fihàn **iye àwọn àkọsílẹ̀ tí a kò tò** rẹ.

## Àwọn iru àkọsílẹ̀

Àwọn àkọsílẹ̀ wá ní **àwọn iru** oríṣiríṣi kí o lè mọ ìmọ̀ láti ìbéèrè láti orísun lójúìgbà — ìpilẹ̀ ètò kan lórí canvas tó ṣòfò láàárín.

## Àwọn ìsopọ̀

Ya **àwọn ìsopọ̀** laarin àwọn àkọsílẹ̀ láti ṣàpẹẹrẹ bíi àwọn ìmọ̀ ṣe jẹmọ́ ara wọn.

## Àwọn awòrán

Fi àwọn awòrán sínú àkọsílẹ̀ láti ìpàpọ̀ àlàyé (tó to **méjì fún àkọsílẹ̀** kọ̀ọ̀kan). Wọ́n ṣe àìgbagbe lórí àwọn ìtúnsọ àti wọ́n lè yọ nígbà èyíkéyìí.

### Wò àwòrán ní ìwọ̀n kíkún

Tẹ àwòrán èyíkéyìí — àwòrán kékeré lórí àkọsílẹ̀, nínú ìpàpọ̀ àlàyé, tàbí àwòrán ńlá lórí kọọpu àwòrán — láti ṣí rẹ̀ ní ìwọ̀n kíkún. Nígbà tí àkọsílẹ̀ bá ní àwòrán jù ọ̀kan lọ, yípadà laarin wọn pẹ̀lú àwọn ọfà tó wà lórí ojú àwòrán tàbí àwọn bọ́tìnnì Òsì àti Ọ̀tún lórí keyboard rẹ. Tẹ **Escape** tàbí tẹ ẹ̀gbẹ́ àwòrán láti pa à.

### Àwọn kọọpu àwòrán

Fún àkọsílẹ̀ tó fi àwòrán sí àkọ́kọ́, fi **àwòrán** kọọpu kan kun láti toolbar lórí canvas. Kọọpu àwòrán fihàn àwòrán ńlá ní àárín dípò ọrọ ara, pẹ̀lú àkọlé gẹ́gẹ́ bí àkọsílẹ̀ àṣàyàn. Kọọpu àwòrán tuntun bẹ̀rẹ̀ pẹ̀lú **àgbègbè fún fífi àwòrán kun** — tẹ rẹ̀ láti yan àwòrán rẹ. Àwọn kọọpu àwòrán gba àwọn awòrán méjì tí wọ́n sì ń sopọ̀ mọ́ àwọn àkọsílẹ̀ mìíràn gẹ́gẹ́ bí kọọpu èyíkéyìí mìíràn.
