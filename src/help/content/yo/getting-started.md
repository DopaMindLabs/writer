# Bẹ̀rẹ̀ sí i

Ibi kíkọ tó jẹ ìbílẹ̀-àkọ́kọ́. Iṣẹ́ rẹ wà nínú **àwọn aaye** tí a sì fipamọ́ rẹ̀ sí aṣàwákiri rẹ fúnra rẹ̀. Kò sí àkọúntì, kò sí cloud, kò sí bọ́tìnnì fipamọ́.

## Ìsọnísókí

1. Láti **Ilé**, ṣí tàbí ṣẹ̀dá aaye kan.
2. Fi ìwé kun láti **ìgbẹ̀yìn**.
3. Bẹ̀rẹ̀ sí tẹ — gbogbo ìtẹ ni a fipamọ́.

Ìyẹn ni gbogbo ìlana. Gbogbo ohun tó wà ní ìsàlẹ̀ jẹ àlàyé tí o lè padà sí nigbakugba.

## Ìwé àkọ́kọ́ rẹ

- Ìwé kan jẹ ibi kíkọ kan nìkan — kò sí àsọyé "fáìlì tuntun".
- Tún sọ orúkọ rẹ̀ nígbà èyíkéyìí nípa tíkẹ lórí orúkọ rẹ̀ nínú **bárì oke**.
- Kò tí sọ orúkọ rẹ̀? Àkọlé ni a yọ láti ìlà àkọ́kọ́ rẹ kí ìgbẹ̀yìn máa ṣe é kà.

## Wíwà ọ̀nà rẹ

- **Ìgbẹ̀yìn** — àwọn aaye, abala, àti ìwé rẹ.
- **Bárì oke** — ìwé lọwọlọwọ àti àwọn tab ìṣàkóso.
- Ọ̀nà **Ìrànlọ́wọ́** nínú ìtọ́nà ojú-ìwé — ń mú ọ padà sí ìbí nígbà èyíkéyìí.

## Àwọn ìrin àjò àtọ́nà

Ìgbà àkọ́kọ́ tí o bá ṣí ojú ìwòye pàtàkì kan, ìrin àjò ìfikọ́ kúkúrú kan ń ṣe àmì àwọn pàtàkì. Fo ìrin àjò kan ní ààrin tí àwọn ìrin àjò yóò dáwọ́ ìbẹ̀rẹ̀ fúnra wọn lórí àwọn ojú ìwòye mìíràn. Tún ṣere ìrin àjò èyíkéyìí nígbà tí o bá fẹ́ láti àkójọ àṣàyàn ìrànlọ́wọ́ (**?**) tàbí [Ìtọ́nà yíyára](customization-and-settings#quick-settings).
