# Ṣíṣe àtòjọ iṣẹ́ rẹ

Ìpele ìrọrùn kan ń pa àwọn iṣẹ́akanṣe mọ́: **àwọn aaye** ní àwọn **abala**, àwọn abala sì ní **àwọn ìwé**.

## Ìsọnísókí

- **Aaye** — iṣẹ́akanṣe kan tó ní àkóónú rẹ̀ (ìwé, ẹ̀kọ́, orúkọ ọ̀rọ̀).
- **Abala** — ẹgbẹ́ àwọn ìwé tó jẹmọ́ ara wọn nínú aaye kan.
- **Ìwé** — ẹyọ tí o ń kọ nínú rẹ̀.
- Bẹ̀rẹ̀ aaye tuntun láti **àdàkọ ètò** láti fo ojú-ìwé ṣòfò.

## Àwọn aaye

Aaye jẹ iṣẹ́akanṣe tó ní àkóónú rẹ̀ pẹ̀lú ìgbẹ̀yìn tirẹ̀, ìtọ́nà, àti àyè [BrainSpace](brainspace). Yípadà àwọn aaye láti ìgbẹ̀yìn tàbí ojú ìwòye **Ilé**.

## Àwọn abala àti ìwé

Nínú aaye kan, ṣe ìpín àwọn ìwé tó jẹmọ́ ara wọn sínú àwọn abala. **Fi kun**, **tún sọ orúkọ**, àti tò àwọn ìwé padà láti ìgbẹ̀yìn. Tẹ **Escape** láti fagilé fikún tàbí títún sọ orúkọ láìfipamọ́.

## Àwọn àdàkọ ètò

Nígbà tí o bá ṣẹ̀dá nǹkan tuntun, bẹ̀rẹ̀ láti **àdàkọ ètò** dípò ojú-ìwé ṣòfò. Àwọn àdàkọ ètò ń ṣe ìkún àwọn abala àti àwọn ìwé bẹ̀rẹ̀ ṣáájú — kí o lè bẹ̀rẹ̀ pẹ̀lú ipèsé tẹ́lẹ̀.

## Títún sọ orúkọ àti ìmọ́tọ̀

- Tẹ orúkọ èyíkéyìí nínú ìgbẹ̀yìn láti tún sọ rẹ̀; **Enter** fún un pa, **Escape** yí padà.
- Pa àwọn ìwé kúkúrú àti tó idojúkọ; gbẹ́kẹ̀lé àwọn abala fún ètò.
- Lo àwọn àkọlé tó kedere kí àwọn àwárí àti ìgbẹ̀yìn máa ṣe é kà.
