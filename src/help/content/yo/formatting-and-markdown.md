# Títọ̀ & markdown

Tọ ọrọ ní ọ̀nà mẹ́ta: toolbar lébèlébè, àwọn kúlẹ̀kúlẹ̀ keyboard, tàbí àwọn kúlẹ̀kúlẹ̀ markdown tí a tẹ bí o ṣe ń kọ.

## Ìsọnísókí

- Tẹ `## ` fún àkọlé, `- ` fún ọdẹ lẹ́tà, `1. ` fún àtòjọ tí a nọ́mbà.
- Dì ọ̀rọ̀ kan pọ̀ pẹ̀lú `**bold**` tàbí `*italic*`.
- Yan ọrọ láti lo toolbar lébèlébè dípò.

## Àwọn kúlẹ̀kúlẹ̀ markdown

Tẹ ìwọ̀nyí ní ìbẹ̀rẹ̀ ìlà kan (tàbí ní ayíká ọ̀rọ̀) tí wọ́n yóò yípadà fúnra wọn:

- `# ` → àkọlé 1, `## ` → àkọlé 2, `### ` → àkọlé 3
- `- ` tàbí `* ` → àtòjọ pẹ̀lú ọdẹ lẹ́tà
- `1. ` → àtòjọ tí a nọ́mbà
- `> ` → ìtúmọ̀ àkójọpọ̀
- `**bold**` → **bold**, `*italic*` → *italic*

## Títọ̀ taara

Yan ọrọ tí o sì lo toolbar lébèlébè, tàbí lo keyboard:

- **Bold** — kúlẹ̀kúlẹ̀ bold
- *Italic* — kúlẹ̀kúlẹ̀ italic

## Àwọn àkọlé àti ètò

Àwọn àkọlé ń ṣèdálẹ̀ **àkójọ àkọlé** ìwé tó farahàn nínú olùyẹ̀wò. Lo wọn láti jẹ kíkọ àwọn ìwé gígùn rọrùn.

## Àwọn àtòjọ àti ìtúmọ̀

Àwọn àtòjọ àti àwọn ìtúmọ̀ àkójọpọ̀ ń pa ìtọ̀nà wọn mọ́ nígbà tí o bá tẹ Enter, nítorí náà o lè tẹ̀síwájú kíkọ láì gba ọwọ́ rẹ fi ẹ̀rọ. Tẹ Enter lórí ìlà àtòjọ tó ṣòfò láti jáde kúrò nínú àtòjọ.
