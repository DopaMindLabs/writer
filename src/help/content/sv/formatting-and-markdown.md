# Formatering och markdown

Formatera text på tre sätt: det flytande verktygsfältet, kortkommandon eller
markdown-genvägar som du skriver medan du arbetar.

## Snabböversikt

- Skriv `## ` för en rubrik, `- ` för en punkt, `1. ` för en numrerad lista.
- Omge ett ord med `**fetstil**` eller `*kursiv*`.
- Markera text för att använda det flytande verktygsfältet istället.

## Markdown-genvägar

Skriv dessa i början av en rad (eller runt ett ord) och de omvandlas automatiskt:

- `# ` → rubrik 1, `## ` → rubrik 2, `### ` → rubrik 3
- `- ` eller `* ` → punktlista
- `1. ` → numrerad lista
- `> ` → blockcitat
- `**fetstil**` → **fetstil**, `*kursiv*` → *kursiv*

## Inline-formatering

Markera text och använd det flytande verktygsfältet, eller använd tangentbordet:

- **Fetstil** — genvägen för fetstil
- *Kursiv* — genvägen för kursiv

## Rubriker och struktur

Rubriker definierar **dispositionen** för dokumentet som visas i inspektören. Använd
dem för att hålla långa stycken navigerbara.

## Listor och citat

Listor och blockcitat behåller sin formatering när du trycker på Enter, så du kan
fortsätta skriva utan att nå efter musen. Tryck Enter på ett tomt listobjekt för
att avsluta listan.
