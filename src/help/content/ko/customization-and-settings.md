# 맞춤 설정

테마, 타이포그래피, 읽기 너비를 취향에 맞게 조정하세요. 페이지 탐색에서
**전체 설정** 화면을 열거나, **빠른 설정**에서 주요 항목을 빠르게 변경할 수 있습니다.

## 한눈에 보기

- **테마** — 밝게, 어둡게, 또는 고대비.
- **타이포그래피** — 본문 폰트와 크기.
- **읽기 너비** — 텍스트 열의 너비.
- **편집기 옵션** — 예: 플로팅 툴바.
- 자주 쓰는 컨트롤은 [빠른 설정](customization-and-settings#quick-settings)에 있습니다.

## 빠른 설정

**빠른 설정** 팝오버를 사용하면 문서를 벗어나지 않고 일상적인 컨트롤에 접근할 수 있습니다.
앱 레일에서 열 수 있습니다. 컨트롤은 목적별로 묶여 있습니다:

- **외관** — **테마** 스위치와 **읽기 너비**.
- **글쓰기** — **포커스 모드**와 **플로팅 툴바** 토글.
- **가이드 투어** — 가이드 투어 다시 보기.
- **더 보기** — 새로운 기능, 접근성, 앱 정보.
- **도움말 센터 →** 링크（**전체 설정 →** 링크와 동일한 방식），도움말과 설정이 항상
  클릭 한 번으로 접근 가능합니다.

## 설정 찾기

전체 설정 화면의 옵션은 **환경 설정**, **글쓰기**, **데이터**, **계정**으로 분류됩니다.
각 그룹은 위에서 아래로 섹션이 쌓인 단일 페이지입니다. 사이드 탐색에서 항목을 선택하면
페이지가 해당 섹션으로 스크롤되고, 스크롤하면서 읽는 섹션이 탐색에서 하이라이트됩니다.

## 테마

**밝게**, **어둡게**, 또는 **고대비** 테마를 선택하세요. 고대비 버전은 가독성과 접근성을 위해
설계되었으며, 하이라이트와 Help Center를 포함한 앱의 모든 부분이 자동으로 적응합니다.
전체 환경 설정 레이어는 [접근성](accessibility)을 참조하세요.

## 타이포그래피

눈에 맞는 본문 폰트와 크기를 조정하세요.

## 읽기 너비

텍스트 열의 너비를 설정하세요. 좁은 너비는 밀도 높은 산문을 읽기 쉽게 만들고, 넓은 너비는
화면에 더 많은 내용을 담습니다.

## 편집기 옵션

작업 방식에 맞게 **플로팅 툴바** 등 편집기 동작을 켜거나 끄세요.

- [내 데이터](your-data) — 작업 내보내기, 가져오기, 백업.
