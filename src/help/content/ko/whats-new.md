# 새로운 기능

LIpsum Writer의 최신 개선 사항을 최신순으로 소개합니다. 전체 변경 로그는
GitHub의 프로젝트 릴리즈 노트를 참조하세요.

## 2026년 6월

- **스마트폰에서의 분할 보기** — 두 패널이 세로 방향에서는 위아래로, 가로 방향에서는
  나란히 배치됩니다. [모바일](mobile) 참조.
- **모바일 문서 검사기** — **더 보기** 탭에서 개요, 정보, 기록, 작업을 열 수 있습니다.
  [문서 검사기](doc-inspector) 참조.
- **건너뛰기를 반영하는 투어** — 가이드 투어를 중간에 닫으면 다른 화면에서도 투어가 자동으로
  시작되지 않습니다. 도움말（**?**）메뉴에서 언제든지 다시 볼 수 있습니다.
- **동기화 상태 한눈에 확인** — 동기화와 백업이 활성화될 때까지 홈 화면에 경고 칩이 표시됩니다.
  [내 데이터](your-data) 참조.
- **문의하기** — **더 보기** 메뉴에서 연락할 수 있습니다. 프로젝트 트래커에 새 이슈가
  열립니다.
