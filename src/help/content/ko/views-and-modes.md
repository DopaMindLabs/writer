# 보기 & 모드

같은 문서를 여러 방식으로 볼 수 있습니다. 상단 바의 탭으로 전환하세요.

## 한눈에 보기

- **글쓰기** — 기본 편집 화면.
- **포커스** — UI 숨기기（**⌘\**）.
- **읽기** — 편집 불가한 깔끔한 읽기 보기.
- **분할** — 두 문서를 나란히.
- **검사기** — 현재 문서의 개요, 정보, 기록.

## 글쓰기

기본 모드: 필요할 때 사이드바와 상단 바를 사용할 수 있는 집중형 편집 화면.

## 포커스

포커스 모드는 주변 UI를 숨겨 텍스트만 남깁니다. **⌘\**（Windows/Linux는 Ctrl+\）로
전환하고, 다시 누르면 UI가 돌아옵니다.

## 읽기

완성된 산문의 편집 불가한 읽기 보기입니다. 독자가 볼 것처럼 초안을 검토할 때 사용하세요.

## 분할

분할 보기는 두 문서를 나란히 표시합니다 — 글을 쓰면서 메모를 참조하거나 두 초안을 비교할 때
유용합니다. 더 큰 화면에서 사용할 수 있습니다.

## 검사기

검사기는 문서의 **개요**, 정보, 기록을 표시합니다. 숨김, 아이콘, 펼침 순으로 크기를 순환합니다.
개요는 제목으로 구성되므로 좋은 제목을 사용하면 즉각적인 탐색이 가능합니다.

## 읽기 너비

좁거나 넓은 너비를 원하나요? [빠른 설정](customization-and-settings#quick-settings)에서
**읽기 너비**를 조정해 눈과 화면에 맞추세요.
