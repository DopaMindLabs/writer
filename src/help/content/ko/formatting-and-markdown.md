# 서식 & 마크다운

텍스트를 서식화하는 방법은 세 가지입니다: 플로팅 툴바, 키보드 단축키, 또는 입력하면서
사용하는 마크다운 단축키.

## 한눈에 보기

- `## ` 로 제목, `- ` 로 글머리 기호, `1. ` 로 번호 목록.
- 단어를 `**굵게**` 또는 `*기울임*` 으로 감싸기.
- 텍스트를 선택해서 플로팅 툴바를 사용할 수도 있습니다.

## 마크다운 단축키

줄의 시작에（또는 단어 주위에）입력하면 자동으로 변환됩니다:

- `# ` → 제목 1, `## ` → 제목 2, `### ` → 제목 3
- `- ` 또는 `* ` → 글머리 기호 목록
- `1. ` → 번호 목록
- `> ` → 블록 인용
- `**굵게**` → **굵게**, `*기울임*` → *기울임*

## 인라인 서식

텍스트를 선택하고 플로팅 툴바를 사용하거나, 키보드를 사용하세요:

- **굵게** — 굵기 단축키
- *기울임* — 기울임 단축키

## 제목과 구조

제목은 검사기에 표시되는 문서의 **개요**를 정의합니다. 긴 글을 탐색하기 쉽게 만드는 데
활용하세요.

## 목록과 인용

목록과 블록 인용은 Enter 키를 눌러도 서식이 유지되므로 마우스 없이 계속 입력할 수 있습니다.
빈 목록 항목에서 Enter 를 누르면 목록을 종료합니다.
