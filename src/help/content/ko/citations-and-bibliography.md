# 인용 & 참고문헌

각 스페이스에는 **BibTeX** 완전 가져오기·내보내기를 지원하는 자체 인용 라이브러리가 있습니다.

## 한눈에 보기

- 참고문헌을 직접 **추가**하거나, `.bib` 파일 / BibTeX 붙여넣기로 **가져옵니다**.
- 항목을 **검색**, **편집**, **삭제**할 수 있으며, 가져올 때 중복은 건너뜁니다.
- **일괄 선택**으로 종류 설정이나 여러 항목을 한 번에 삭제할 수 있습니다.
- 전체 라이브러리를 `.bib`로 **내보낼** 수 있습니다.

## 참고문헌 가져오기

BibTeX를 붙여넣거나 가져오면 항목이 스페이스의 라이브러리에 추가됩니다. 파서는 표준
BibTeX 필드와 **다양한 항목 유형**을 읽고, 추가된 수와 **중복**으로 건너뛴 수를 보고합니다.

## 라이브러리 관리

- 인용 패널에서 항목을 탐색하고 **검색**합니다. **지우기** 버튼을 누르면 전체 목록으로
  돌아갑니다.
- 항목을 열어 **세부 보기**를 표시하고, **편집**으로 전환해 인라인에서 저장합니다.
- 행별로 **일괄 선택**하거나 전체 선택해서 확인 대화 상자를 통해 종류를 설정하거나 삭제합니다.

## 내보내기

라이브러리를 BibTeX로 언제든지 내보내 다른 곳에서 사용하거나 산문 옆에 백업으로 보관하세요.
전체 내보내기 옵션은 [내 데이터](your-data)를 참고하세요.

## 키보드

인용 패널 내에서:

- **Esc** — 패널 닫기（또는 진행 중인 편집 취소）.
- **Ctrl/⌘ + Enter** — 편집 중인 항목 저장.
