# 전체 기능

앱이 할 수 있는 모든 것을 한눈에 살펴보는 목록입니다. 각 항목의 링크를 클릭하면
해당 가이드로 바로 이동합니다.

## 글쓰기 & 편집

- **자동 저장** — 모든 키 입력이 로컬에 저장되며 저장 버튼이 없습니다. [자세히 보기](writing-and-editing#autosave)
- **플로팅 툴바** — 텍스트 선택 시 빠른 서식 적용. [자세히 보기](writing-and-editing#the-floating-toolbar)
- **문서 이름 바꾸기** — 상단 바의 제목 클릭. [자세히 보기](writing-and-editing#renaming-a-document)
- **영속성** — 강제 새로고침 후에도 초안이 유지됩니다. [자세히 보기](your-data#what-local-first-means)

## 서식

- **마크다운 단축키** — 입력하면서 제목, 목록, 인용, 굵게/기울임 적용. [자세히 보기](formatting-and-markdown#markdown-shortcuts)
- **제목이 개요를 구성** — 긴 글 구조화. [자세히 보기](formatting-and-markdown#headings-and-structure)

## 정리

- **스페이스** — 독립적인 프로젝트. [자세히 보기](organizing-your-work#spaces)
- **섹션 & 문서** — 사이드바에서 추가, 이름 변경, 순서 변경. [자세히 보기](organizing-your-work#sections-and-documents)
- **템플릿** — 빈 페이지가 아닌 구조에서 시작. [자세히 보기](organizing-your-work#templates)

## 보기 & 모드

- **글쓰기** — 기본 편집 화면. [자세히 보기](views-and-modes#write)
- **포커스 모드** — ⌘\ 로 UI 숨기기. [자세히 보기](views-and-modes#focus)
- **읽기** — 편집 불가한 깔끔한 읽기 보기. [자세히 보기](views-and-modes#read)
- **분할** — 두 문서를 나란히. [자세히 보기](views-and-modes#split)
- **검사기** — 개요, 정보, 기록. [자세히 보기](views-and-modes#the-inspector)

## 인용

- **BibTeX 가져오기** — `.bib` 붙여넣기 또는 업로드（중복 건너뛰기 포함）. [자세히 보기](citations-and-bibliography#importing-references)
- **항목 관리** — 검색, 인라인 편집, 일괄 선택, 삭제. [자세히 보기](citations-and-bibliography#managing-the-library)
- **`.bib`로 내보내기** — 라이브러리를 어디서나 사용. [자세히 보기](citations-and-bibliography#exporting)

- **연결** — 메모들의 관계 지도 만들기. [자세히 보기](brainspace#connections)

- **전체 화면 뷰어** — 이미지를 클릭해 전체 화면으로. [자세히 보기](brainspace#view-a-picture-full-size)
- **이미지 카드** — 툴바에서 이미지 중심 메모 만들기. [자세히 보기](brainspace#image-cards)
