# 문서 검사기

문서 검사기는 편집기 옆의 패널입니다. **개요** 탭은 문서의 구조를 보여주고, **정보** 탭은
작성 중인 문서에 관한 실시간 세부 정보를 표시하며 몇 가지 선택적 제한과 상태를 설정할 수
있습니다. 표시할 필드를 전체적으로 또는 스페이스별로 선택할 수 있어 검사기에는 필요한
정보만 표시됩니다.

## 개요

**개요** 탭은 문서의 제목을 레벨에 따라 들여쓰기하여 순서대로 나열하므로, 긴 글의 구조를
한눈에 파악할 수 있습니다. 입력을 멈춘 직후 업데이트됩니다.

개요는 제목으로 구성됩니다 — 줄 시작에 `# `, `## ` 또는 `### ` 을 입력하거나（플로팅
툴바 사용）제목을 추가하세요. 제목이 없는 문서는 빈 개요를 표시합니다. 제목 사용법은
[서식 & 마크다운](formatting-and-markdown#headings-and-structure)을 참조하세요.

스마트폰에서는 **더 보기** 탭에서 검사기를 엽니다. 메뉴에서 **문서 검사기**를 선택하면
오른쪽에서 슬라이드인됩니다.

## 단어 수와 문자 수

정보 탭에는 항상 문서의 실시간 **단어 수**와 **문자 수**가 표시됩니다.

## 단어 수 또는 문자 수 제한 설정

**단어 제한** 또는 **문자 제한** 필드가 표시될 경우 숫자를 입력해 목표를 설정하세요.
카운터는 `현재 / 제한` 형식으로 표시되며, 제한을 초과하면 숫자가 빨간색으로 변하고
편집기에서 제한 초과 텍스트가 하이라이트됩니다. 삭제나 잘라내기는 일어나지 않습니다 —
하이라이트는 시각적 단서일 뿐이므로 계속 작성하고 나중에 정리할 수 있습니다.

제한 없이 사용하려면 제한 필드를 비워두거나 `0`으로 설정하세요.

제한과 카운터는 유지하되 편집기의 하이라이트는 원하지 않는다면 설정에서
**제한 초과 텍스트 하이라이트**를 끄세요（아래 참조）.

## 상태와 잠금

**상태** 선택기로 문서를 각 단계로 이동시킬 수 있습니다: _초안_, _진행 중_, _검토 중_,
_완료_, _게시됨_. 상태를 **완료** 또는 **게시됨**으로 설정하면 실수로 변경되지 않도록
문서가 잠기고, 편집기 위에 **편집하려면 잠금 해제** 버튼이 있는 배너가 표시됩니다.
잠금을 해제하려면 해당 버튼을 사용하거나 상태를 이전 단계로 되돌리세요. 잠금은 문서 본문만
보호하며 아무것도 삭제되지 않습니다.

## 마감일

**마감일** 필드가 표시될 경우 날짜를 선택해 기한을 기록하세요. 기한이 지난 날짜는 빨간색으로 표시됩니다.

## 표시할 필드 선택

모든 검사기 필드는 선택적입니다. 표시할 필드를 선택하려면:

- **전체적으로:** **전체 설정 → 문서 검사기**를 열고 각 필드를 켜거나 끕니다. 상태 선택기에
  표시할 단계도 선택할 수 있습니다.
- **스페이스별로:** 스페이스의 **설정 → 문서 검사기**를 엽니다. 각 필드는 전체 기본값을
  상속하거나 해당 스페이스에서만 켜거나 끌 수 있습니다.

필드를 끄면 제한값과 편집기 하이라이트를 포함해 검사기에서 숨겨집니다 — 이전에 값을
설정했더라도 마찬가지입니다. 값은 삭제되지 않습니다. 필드를 다시 켜면 다시 표시됩니다.
