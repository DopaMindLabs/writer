# 시작하기

로컬 우선 방식의 글쓰기 공간입니다. 작업은 **스페이스**에 저장되고 브라우저에 자동으로
저장됩니다. 계정, 클라우드, 저장 버튼이 필요 없습니다.

## 한눈에 보기

1. **홈**에서 스페이스를 열거나 만드세요.
2. **사이드바**에서 문서를 추가하세요.
3. 입력을 시작하세요 — 모든 키 입력이 저장됩니다.

이것이 전체 흐름입니다. 아래의 내용은 나중에 돌아와 읽을 수 있는 세부 사항입니다.

## 첫 번째 문서

- 문서는 글을 쓰는 공간일 뿐입니다 — "새 파일" 대화 상자가 없습니다.
- **상단 바**에서 이름을 클릭하면 언제든지 이름을 바꿀 수 있습니다.
- 이름을 지정하지 않으면 첫 번째 줄에서 제목이 자동으로 생성됩니다.

## 화면 구성

- **사이드바** — 스페이스, 섹션, 문서.
- **상단 바** — 현재 문서와 모드 탭.
- 페이지 탐색의 **도움말** 링크 — 언제든지 여기로 돌아올 수 있습니다.

## 가이드 투어

주요 화면을 처음 열면 짧은 인터랙티브 투어가 핵심을 안내합니다. 투어를 중간에 건너뛰면
다른 화면에서도 투어가 자동으로 시작되지 않습니다. 도움말（**?**）메뉴 또는
[빠른 설정](customization-and-settings#quick-settings)에서 언제든지 투어를 다시 볼 수 있습니다.
