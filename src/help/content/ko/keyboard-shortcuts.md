# 키보드 단축키

키보드에서 손을 떼지 않고 작업하세요. macOS는 **⌘**, Windows와 Linux는 **Ctrl** 을 사용합니다.

## 한눈에 보기

- **⌘K** — 어디서나 빠른 도움말 열기.
- **⌘\** — 포커스 모드 켜기/끄기.
- **Esc** — 오버레이 닫기 또는 편집 취소.
- **Ctrl/⌘ + Enter** — 편집 중인 인용 저장.

## 전체 공통

- **⌘K** — 어디서나 **빠른 도움말** 오버레이 열기. Help Center를 검색하거나 단축키로
  이동하고, **Esc** 로 닫습니다.
- **⌘\** — [포커스 모드](views-and-modes#focus) 켜기/끄기.

## 편집기 마크다운

줄 시작에 입력하면 글을 쓸 때 변환됩니다:

- `# ` / `## ` / `### ` — 제목
- `- ` / `* ` — 글머리 기호 목록
- `1. ` — 번호 목록
- `> ` — 블록 인용

단어를 `**` 로 감싸면 굵게, `*` 로 감싸면 기울임이 됩니다.

## 인용 패널

- **Esc** — 패널 닫기（또는 진행 중인 편집 취소）
- **Ctrl/⌘ + Enter** — 편집 중인 항목 저장

## 브라우저 단축키에 대하여

일부 키 조합은 브라우저나 운영체제에서 예약되어 있습니다. 앱이 단축키를 정의한 경우
해당 화면에서 우선 적용되고, 그 외의 곳에서는 시스템 기본값이 적용됩니다.
